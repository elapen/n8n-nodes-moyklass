1️⃣  Установите зависимости:
    cd /Users/bekzat/Documents/projects/n8n_moyklass
    npm install

2️⃣  Соберите проект:
    npm run build

3️⃣  Протестируйте локально:
    npm link
    cd ~/.n8n
    npm link n8n-nodes-moyklass
    n8n restart

4️⃣  Откройте в n8n и протестируйте

5️⃣  Опубликуйте в npm:
    npm login
    npm publish

6️⃣  Создайте GitHub репозиторий:
    git init
    git add .
    git commit -m "Initial commit"
    git remote add origin https://github.com/USERNAME/n8n-nodes-moyklass.git
    git push -u origin main
